# Layered Relief Art App

## Admin Workflow

1. **Log in** to the admin dashboard.
2. **Upload City**: add a skyline photo, city name, and select a style, then click **Add to Queue**.
3. **Stage 1 (Style Transfer)**: in **Processing Queue**, click **Run Stage 1** to generate the line-art SVG.
4. **Adjust Spacing**: open the spacing panel, set the horizontal expansion, and click **Apply Spacing**.
5. **Stage 2 (Layer Separation)**: click **Run Stage 2** to generate three SVG layers.
6. **Download**: open **City Bank** (or **View Result**) and download the layer files.

For the full workflow, prompts, and API details, see [`DOCUMENTATION.md`](./DOCUMENTATION.md).
